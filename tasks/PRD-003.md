# PRD-003 — Training Pipeline Bootstrap

## Goal
Prepare reproducible training launch path using reference code and validated assets.

## Inputs
- reference train entry: `repositories/RGBT-Crowd-Counting/train.py`
- validated dataset and checkpoint

## Steps
1. Define canonical train command with resolved paths.
2. Set run directory for checkpoints/logs.
3. Execute smoke training window (short run) for command correctness.
4. Lock command template in runbook.

## Outputs
- canonical train command.
- checkpoint/log location policy.

## Acceptance Criteria
1. Train command starts without import/path errors.
2. Checkpoint directory is writable and receives output.
